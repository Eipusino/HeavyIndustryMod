## 基本信息

**基于原版的内容拓展mod。 也可以作为隐藏mod不加载内容仅作为开发前置库，提供多种工具和方法。**

**当前mod处于开发阶段，大多数API极其不稳定，可能随时在未来的更新中变更，暂时不建议作为依赖源。**

+ **模组版本号: 1.0.9-beta**
+ **模组类型: Java**

---

## 运行环境

**游玩该模组前请确保你的设备环境符合模组运行环境要求。否则模组无法正常运行。**

+ **最小Mindustry版本要求: 155.0**
+ **{桌面、Linux与MAC OS} 最小Java版本要求: 16**
+ **{安卓} 安卓平台。最小安卓版本要求: 9（API等级28）**
+ **不支持IOS**
